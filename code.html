Biographs

Biographs is a Python module. It converts protein structural file formats like .pdb and .cif into amino acid networks. Using biopython package, you can calculate distance, iterate over nodes and atoms and measure many other structural features. Additionally, it contains implementations of algorithms measuring the void or emptyness around amino acids in the structure.
